# Classifying Player Behavior in Video Games Using Logistic Regression
Using logistic regression to predict player behavior or play style as aggressive or passive.

The repo contains a Jupyter Notebook file along with a csv that is used for the project. The csv is a dataset of player gameplay telemetry from the video game PUBG that contains 5,000 rows of data.

Anaconda environment was used to write and run the Jupyter Notebook file provided in the repo.

Libraries used:
os,
pandas,
numpy,
sklearn.metrics,
sklearn.preprocessing,
sklearn.model_selection,
sklearn.linear_model,
seaborn,
matplotlib.pyplot
